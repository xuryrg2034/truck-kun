# Economy Feature

Система денег и наград.

## Системы

| Система                 | Назначение     |
| ----------------------- | -------------- |
| MoneyCollectionSystem   | Сбор денег     |
| RewardCalculationSystem | Расчёт награды |

## Компоненты

- `Money` (Meta) — текущий баланс
- `MoneyEarned` — заработано за день
- `Reward` — награда за действие

## Сервисы

- [[MoneyService]] — управление балансом

## Зависимости

- **Получает от:** [[Collision]] — награды за попадания
- **Используется:** [[UpgradeService]] — покупка улучшений
- **Используется:** UI — отображение баланса

## Экономический баланс

| Источник | Награда |
|----------|---------|
| Студент | 100 ¥ |
| Бабушка | 200 ¥ |
| Офисный работник | 150 ¥ |
| Бонус скорости | x1.5 |
