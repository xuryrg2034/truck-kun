# Player Progression

Прогрессия игрока между днями.

## Цикл прогресса

```
День N → Заработок → Улучшения → День N+1 (сложнее, но сильнее)
```

## Системы прогресса

| Система | Влияние |
|---------|---------|
| [[UpgradeService]] | Улучшение характеристик |
| [[QuestService]] | Бонусные награды |
| Difficulty Scaling | Увеличение сложности |

## Улучшения

Игрок покупает улучшения в [[UpgradeService]]:

- **Speed** — быстрее едет, больше успевает
- **Acceleration** — быстрее разгоняется
- **Handling** — лучше управляется
- **Armor** — больше прочность

## Квесты

Ежедневные квесты дают бонусные награды:

- "Сбить 5 студентов" — +500 ¥
- "Сбить 3 бабушки" — +800 ¥

## Difficulty Scaling

С каждым днём:

- Больше пешеходов
- Быстрее они двигаются
- Больше препятствий
