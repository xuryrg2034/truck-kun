# Настройка префабов препятствий

## Общие правила

1. Все префабы должны иметь Layer: **Obstacle** (10)
2. Добавить компонент `ObstacleBehaviour` и выбрать тип
3. Разместить в `Assets/Prefabs/Obstacles/`

---

## Ramp (Трамплин)

**Описание:** Наклонная поверхность, подбрасывающая Hero вверх через физику.

### Настройка:
1. Создать 3D объект (Cube или Mesh)
2. Повернуть на 15° по оси X (наклон к игроку)
3. Добавить **MeshCollider** (Convex: true)
4. **НЕ добавлять Rigidbody** (статический объект)
5. Добавить `ObstacleBehaviour`, выбрать Kind: **Ramp**

### Примерные размеры:
- Ширина: 3-4 м (перекрывает полосу)
- Длина: 2-3 м
- Высота подъёма: 0.5-1 м

```
    /|
   / |  ← Наклон 15°
  /__|
```

---

## Barrier (Барьер)

**Описание:** Тяжёлый объект, который можно сдвинуть. Замедляет Hero при столкновении.

### Настройка:
1. Создать 3D объект (Cube)
2. Добавить **BoxCollider**
3. Добавить **Rigidbody**:
   - Mass: **200 kg**
   - Drag: 2
   - Angular Drag: 1
   - Use Gravity: true
   - Is Kinematic: false
4. Добавить `ObstacleBehaviour`, выбрать Kind: **Barrier**

### Примерные размеры:
- Ширина: 1.5-2 м
- Высота: 1 м
- Глубина: 0.5 м

---

## SpeedBump (Лежачий полицейский)

**Описание:** Небольшое возвышение. Даёт лёгкий подброс и немного замедляет.

### Настройка:
1. Создать 3D объект (вытянутый полуцилиндр или Cube)
2. Добавить **BoxCollider** или **MeshCollider**
3. **НЕ добавлять Rigidbody** (статический)
4. Добавить `ObstacleBehaviour`, выбрать Kind: **SpeedBump**

### Примерные размеры:
- Ширина: 4-6 м (вся дорога)
- Высота: 0.1-0.15 м
- Глубина: 0.3-0.5 м

---

## Hole (Яма)

**Описание:** Триггер-зона. Тянет вниз и сильно замедляет.

### Настройка:
1. Создать 3D объект (Cube или Plane для визуала)
2. Добавить **BoxCollider**:
   - **Is Trigger: true** ← Важно!
3. **НЕ добавлять Rigidbody**
4. Добавить `ObstacleBehaviour`, выбрать Kind: **Hole**
5. Опционально: добавить визуал ямы (mesh с текстурой)

### Примерные размеры:
- Ширина: 2-3 м
- Глубина (по Z): 3-4 м
- Высота триггера: 1-2 м (чтобы Hero попадал)


---

## Размещение на уровне

1. Перетащить префаб на сцену
2. Разместить на дороге (Y = 0 для ground level)
3. Убедиться что Layer = Obstacle
4. Для Barrier: можно дублировать и создавать группы

---

## Тестирование

После создания префабов:
1. Запустить сцену Battle
2. Проверить что Hero взаимодействует с каждым типом:
   - Ramp: подбрасывает
   - Barrier: сдвигается, Hero замедляется
   - SpeedBump: лёгкий подброс
   - Hole: тянет вниз, замедляет
